What is HTML

HTML stands for Hyper Text Markup Language ,it is the standard markup language for creating Web pages and describes the structure of a Web page.

What is CSS?

CSS is the language we use to style an HTML document. It describes how HTML elements are to be displayed on screen, paper, or in other media and saves a lot of work. It can control the layout of multiple web pages all at once.

what is JS?

JavaScript is the Programming Language for the Web.It can update and change both HTML and CSS. JavaScript can calculate, manipulate and validate data.
